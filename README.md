# Portfolio

Just a small, clean portfolio website built with SvelteKit for myself. Might grow into something more in the future, you never know.

## Tech stack
- SvelteKit
- TypeScript
- Vite
- Tailwind

## Deployment
The `main` branch is used to deploy the site using Vercel at [ivaniliev.com](https://www.ivaniliev.online), whereas the `develop` branch is used for the development.

## Running locally
```bash
npm install
npm run dev
```
